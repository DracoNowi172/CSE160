Notes to Grader

The instructions were ambiguous when it came to the image button so I interpereted it as "as long as the image is shown on screen and the other painting features dont crash its alright". Consequentially I felt that my masterpiece of an image needed its own canvas so I set it so that the image would delete the other points before being printed onto the canvas. Also the image color can be changed, it takes the last color printed on screen before the image is printed. I preffer unifrom colors so I designed it this way since the Aegis Trinity processor core crystals can come in different colors in the game based on the last used art. So I figured it would be a good easter egg.
